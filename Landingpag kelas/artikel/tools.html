Workflow Recommendations

Network Documentation Workflow:

  1. Draft: Sketch di paper atau Miro
  2. Diagram: Create di Draw.io
  3. Enhance: Add labels dan annotations
  4. Export: SVG untuk scalability, PNG untuk sharing
  5. Document: Embed in Confluence atau Notion

UI/UX Design Workflow:

  1. Research: User needs, competitors
  2. Wireframe: Low-fi sketches di Figma
  3. Design: High-fi mockups dengan real content
  4. Prototype: Add interactions
  5. Test: User testing dan feedback
  6. Handoff: Developer handoff via Figma

Presentation Workflow:

  1. Content: Outline di Notion atau Docs
  2. Visuals: Create graphics di Canva
  3. Diagrams: Technical diagrams di Draw.io
  4. Assembly: Combine di PowerPoint atau Google Slides
  5. Polish: Animations dan transitions
